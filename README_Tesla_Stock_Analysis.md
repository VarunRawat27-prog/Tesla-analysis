# Tesla Stock Analysis

## Project Overview

This project analyzes historical Tesla stock market data using Python.
It demonstrates how to load, clean, analyze, and visualize stock data
with Pandas, NumPy, Matplotlib, and Seaborn to extract meaningful
insights.

------------------------------------------------------------------------

## Objective

Practice the following concepts:

-   Data Loading
-   Data Cleaning
-   Statistical Analysis
-   Data Filtering
-   Data Visualization
-   Correlation Analysis

------------------------------------------------------------------------

## Technologies

-   Python 3
-   Jupyter Notebook / Google Colab

### Libraries Used

-   NumPy
-   Pandas
-   Matplotlib
-   Seaborn

------------------------------------------------------------------------

## Dataset

-   **TSLA.csv** (Historical Tesla Stock Data)

------------------------------------------------------------------------

## Features

-   Load the Tesla stock dataset.
-   Display dataset information.
-   Check data types and missing values.
-   Calculate summary statistics.
-   Convert Date to DateTime format.
-   Filter stock records using conditions.
-   Find top and bottom stock values.
-   Create line plots, bar plots, histograms, heatmaps, and scatter
    plots.
-   Perform correlation analysis.

------------------------------------------------------------------------

## How to Run

1.  Install the required libraries:

``` bash
pip install numpy pandas matplotlib seaborn
```

2.  Place `TSLA.csv` in the same folder as the notebook.
3.  Open the notebook in Jupyter Notebook or Google Colab.
4.  Run all cells sequentially.
